# Rewrite Communication

This skill rewrites emails, messages, and other communications into professional, regionally-appropriate formats.

## Purpose

Transform draft communications into polished, professional messages that:
- Maintain the original intent and key information
- Use appropriate regional spelling and conventions
- Strike the right professional tone
- Include properly formatted signature blocks

## Usage

When asked to rewrite an email or message:

1. **Analyse the draft**: Understand the core message, audience, and purpose
2. **Apply regional conventions**: Use appropriate spelling, date formats, and cultural tone
3. **Enhance clarity**: Improve structure and readability without changing meaning
4. **Add signature**: Include professional signature block with provided details

## Signature Block Format

Include when personalisation details are provided:

```
[Sign-off],

{{FULL_NAME}}
{{JOB_TITLE}}
{{ORGANISATION}}
{{PHONE}}
{{EMAIL}}
{{PRONOUNS}}
```

Omit any fields not provided. Format phone numbers according to regional conventions.

## Tone Guidelines

### Professional Contexts
- Clear and direct subject lines
- Structured paragraphs with logical flow
- Professional but not overly formal
- Action items clearly stated

### Sensitive Communications
- Empathetic and respectful tone
- Careful word choice
- Clear but considerate phrasing

### Follow-up Messages
- Reference previous correspondence appropriately
- Maintain consistent tone with thread
- Clear next steps or requests

## Output Format

Always present rewritten communications in an artifact with:
- Clear subject line (if email)
- Properly structured body
- Appropriate signature block
- No meta-commentary unless requested

## Examples

**Original draft:**
"hey can u send me that report we talked about? need it asap thx"

**Rewritten (NZ style):**
```
Subject: Request for Report

Hi [Name],

Could you please send through the report we discussed? I'd appreciate 
receiving it as soon as possible.

Thanks,
[Your name]
```

The rewrite maintains the request while adding professional structure and clarity.
